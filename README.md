## 🛠️ عن المنصة
**سحورنا 2027** هي منصة ويب تفاعلية (PWA) تم تصميمها لتكون الرفيق الرقمي للمسلم خلال شهر رمضان المبارك. تجمع المنصة بين دقة البيانات الدينية وجمال التصميم العصري (Cyber-Ramadan Style) مع ميزات تقنية متقدمة مثل متابعة الزوار لحظياً وتحديد المواقع التلقائي.

---

## 👥 فريق العمل (المطورون)
تم بناء هذا العمل بجهد جماعي من فريق متميز:
* **كريم عمر**: المطور الأساسي والمصمم البصري (Full Stack Developer).
* **مصطفى محمد (عبقرينو)**: شريك التطوير والبرمجة.
* **محمد بركات**: تطوير وإدارة المحتوى.
* **يوسف عويس**: مراجعة البيانات والخدمات.
* **يوسف مصطفى**: تنسيق واجهة المستخدم.

---

## ✨ الخدمات والميزات
تنقسم المنصة إلى قسمين رئيسيين لضمان تغطية كافة احتياجات المستخدم:

### 🕋 الخدمات الدينية
* **مواقيت الصلاة**: عرض دقيق حسب الموقع الجغرافي للمستخدم.
* **المصحف الشريف**: قراءة القرآن الكريم بواجهة مريحة.
* **أذكار المسلم**: مجموعة شاملة من الأذكار اليومية.
* **المسبحة الإلكترونية**: عداد تفاعلي للتسبيح.
* **متابعة الختمة**: أداة ذكية لتنظيم ختم القرآن.
* **حاسبة الزكاة وفتاوى الصيام**: مركز تعليمي وديني متكامل.

### 🍕 جودة الحياة والترفيه
* **وصفات الفطور وسحور**: نصائح تغذية يومية.
* **جدول المسلسلات**: متابعة مواعيد البرامج والمسلسلات.
* **ركن الأطفال**: محتوى تعليمي مخصص للصغار.
* **متبع العادات**: أداة لتحسين العادات اليومية في رمضان.

---

## 🚀 التقنيات المستخدمة
* **Frontend**: HTML5, CSS3 (Glassmorphism), JavaScript (ES6+).
* **Backend**: Firebase Realtime Database (لتتبع الزيارات والزوار المتصلين).
* **APIs**: Aladhan API لمواقيت الصلاة والتاريخ الهجري.
* **Optimization**: PWA (Progressive Web App) لإمكانية تثبيت الموقع كتطبيق على الهاتف.
